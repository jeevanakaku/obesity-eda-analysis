# Obesity EDA Analysis 🔍

## About the Project
Exploratory Data Analysis on an obesity dataset containing 2000+ records
to identify correlations between lifestyle habits and obesity levels.

## Tools & Technologies
- Python (Google Colab)
- Pandas — data cleaning & manipulation
- Matplotlib & Seaborn — data visualization
- NumPy — numerical operations

## What I Did
- Cleaned the dataset — handled missing values and removed duplicates
- Analyzed correlations between lifestyle habits and obesity levels
- Visualized patterns using charts and graphs
- Found that exercise and diet habits have strong correlation with obesity

## Key Finding
Lifestyle habits like exercise frequency and eating patterns showed 
strong correlation with obesity levels in the dataset.

## Dataset
2000+ records with features like age, gender, eating habits, 
physical activity, and obesity classification.
